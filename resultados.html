Introdução

Foram criadas duas versões, uma sequencial e uma concorrente de um programa que faz N requisições à API Roll Dice (https://rapidapi.com/vishukaypri/api/roll-dice1) e armazena os resultados obtido dessas rolagem virtual.

Metodologia

Para cada versão do programa foi executado um conjunto de 2, 4, 8, 16, 32, 64, 128, 256, 512 requisições, cada um 20 vezes, anotando-se o tempo decorrido máximo, mínimo, a média e o desvio padrão.

Para o presente experimento foi utilizada a seguinte máquina:

Processador gráfico: Mesa Intel® Xe Graphics
